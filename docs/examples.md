# Chatlet Examples (実用プリセットレシピ集)

Chatlet Free は、用途に特化したプリセット（配列）を差し替えるだけで、専門分野にパーソナलाइजされた特製コントローラーへと進化します。以下をコピーして、そのまま `presets` プロパティへ適用してください。

## ✍️ 推敲・翻訳パック (Writer Pack)
```json
[
  { "label": "校正", "value": "誤字脱字、不自然な表現を直して：\n\n" },
  { "label": "要約", "value": "内容を崩さず極限までシンプルに縮めて：\n\n" },
  { "label": "敬語", "value": "丁寧なビジネス用の敬語に書き直して：\n\n" }
]
```

## 💻 開発・デバッグパック (Developer Pack)
```json
[
  { "label": "レビュー", "value": "コードのバグやリファクタ点がないかレビューして：\n\n" },
  { "label": "テスト", "value": "このコードに対するUnitテストケースを作成して：\n\n" },
  { "label": "バグ修正", "value": "以下のクラッシュや例外の原因を分析して直して：\n\n" }
]
```

## 💡 壁打ち・企画パック (Critical Thinking Pack)
```json
[
  { "label": "問題指摘", "value": "このアイデアに対するボトルネックや論理の穴を教えて：\n\n" },
  { "label": "反対意見", "value": "私を説得するように、この提案への痛烈な反対意見を述べて：\n\n" }
]
```
